# Installation

Clone the repository or download the plugin files.

```bash
git clone https://github.com/yourusername/ai-summarizer-plugin.git
```

Navigate to your WordPress installation and move the plugin folder into the wp-content/plugins/ directory.
Go to your WordPress admin dashboard and activate the AI Summarizer plugin in the Plugins section.
Optionally, install the required dependencies via Composer (if not already installed):

```bash
composer install
```
